# CT-AI System for Mammography Analysis

This project is a diploma work focused on AI-assisted mammography analysis.

## Features
- FastAPI backend for AI inference
- YOLO-based lesion detection
- Lesion classification
- Clinical risk estimation
- React frontend for visualization

## Project structure
backend/ - FastAPI backend
frontend/ - React frontend
datasets/ - Not included (due to size)
models/ - Not included (due to size)

## Models & datasets
Model weights and datasets are not stored in this repository due to size limits.

They must be provided separately.

## Disclaimer
This project is for research and educational purposes only and is not a medical device.
